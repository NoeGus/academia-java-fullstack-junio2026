# spring-batch-final-calificaciones

Proyecto Spring Batch para calificaciones de estudiantes.

## Qué incluye

- Job de Spring Batch con 2 pasos.
- Persistencia en MySQL y MongoDB.
- API REST para consultar y modificar estudiantes.
- Pruebas unitarias de processors, service y controlador REST.

## Ejecutar pruebas

```bash
mvn test
```
